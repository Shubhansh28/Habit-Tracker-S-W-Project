# Health Streaks

A simple fitness tracker app built with HTML, CSS, and JavaScript. Track workouts, stats, goals, and BMI!

## Features
- Add workouts with categories (Cardio, Strength, etc.)
- View stats and history
- Set weekly goals with progress bars
- Calculate BMI
- Streak counter for motivation
